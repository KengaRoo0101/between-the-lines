Data Retention

This page explains what data may remain temporarily in the current Between The Lines implementation and what does not.

Uploaded files

Uploaded files are used to generate your report and are not written to disk as part of report generation.

Report state

Generated report data may remain in your browser session temporarily so the report can stay available while you review it.

Payment state

Payment verification status may remain temporarily in server memory to confirm whether a checkout session has completed.

Optional research consent

Optional anonymized pattern data may be used only if you consent. Private messages and identifying details are not sold.
