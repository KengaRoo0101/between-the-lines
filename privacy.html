How uploaded files are used

Uploaded files are used to generate the report you request.

We do not sell private uploaded conversations.

Research consent

Optional anonymized pattern data may be used only if you affirmatively consent. Private messages and identifying details are not sold.

Storage and retention

Uploaded files are processed to generate your report and are not written to disk as part of report generation.

Generated report data may remain temporarily in your browser session so you can read the report after it is created. Temporary server-side payment state may remain in memory for the active runtime only.

Payment is processed by Stripe. We do not store card data.
