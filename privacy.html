Privacy

This page describes how uploaded files are handled in this implementation of Between The Lines.

Processing

Uploaded files are processed transiently by the local app server during the current session.

Files are analyzed only to generate the report requested by the user.

Storage

This implementation does not write uploaded files to disk as part of report generation.

Sharing

This implementation does not share uploaded files with third parties as part of report generation.
